# يوسف بوت 🤖

بوت تليجرام لتخزين الملفات بشكل آمن.

## 🚀 التشغيل السريع

```bash
# 1. نسخ المستودع
git clone https://github.com/skwrbo-debug/Youssef-Bot.git
cd Youssef-Bot

# 2. إعداد المتغيرات
cp .env.example .env
# عدّل .env بمفاتيحك الخاصة

# 3. التثبيت
pip install -r requirements.txt

# 4. التشغيل
python storage_bot_secure.py
